Presentation template help


Use the Text field to add the text you wish to appear on each page. You can highlight text and right click to add basic html or type the code directly in the text box such as <b>bold text</b> for bold text or <a href="www.google.com">hyperlinks</a> for hyperlinks. The html will appear red in the editor.

Special note: to use layout 8 with 2 text blocks add the custom tag <split> to determine where the text is split and displayed in the second text block. e.g. <split>This is where the second text block text begins...

 


Here you can easily change the default font size for each page. You can also use html to change font properities within the text block.

 


Use the Learning Object Title field to add a main title for your learning object

 


Add a title for each of your pages in the page title field

 


Use the browse and preview icons to select an image (jpg or png), swf or flv file to display on any page

 


Use the Alt text field to add an accessible description of your images and movies. This will also appear as a tooltip on images

 


Use the layout menu to choose one of the preset layouts shown below
Layout 1 Text only full page
Layout 2 Text left wide image right
Layout 3 Text right wide image left
Layout 4 Text left tall image right
Layout 5 Text right tall image left
Layout 6 Full page image caption right
Layout 1 showing flash movie
Layout 1 showing flv movie
Layout 7 full screen image/swf/flv
 
Layout 8 2 text blocks 2 images/movies
Layout 9 Relative text box
  

 


If you wish to prevent your selected images from resizing according to the selected layout tick this option

 



Use the override height and width fields to manually override the layout settings for image/movie size. Be careful to use dimensions proportional to your original image/movie size or distortion may occur.

 


Use the sound menu to select an optional audio (mp3) file for each page

 


Use this option to select a second image or movie for layout 8

 


If you add a second image or movie be sure to add the alt text description here

 


You can add a web link or link to a local file for each page here. Wherever you have added a link an icon will appear at the bottom of the screen.

 


If you add a link add the alt text and tool tip here
